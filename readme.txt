== Description ==

<h3>MK WordPress Block</h3>

This powerful Free Plugin helps you to add blocks in your theme and enhance your website functionality, you can add block in your post, page, widget, custom post type. It works with wordpress editor with <strong>TinyMCE Shortcode</strong> list and also with <strong>Visual Composer</strong> by element.

= Plugin Features =

1. Works with wordpress editor.
2. Works with <strong>Visual Composer</strong>.
3. Add Block in post.
4. Add Block in page.
5. Add Block in widget.
6. Add Block in custom post type.
7. Shortcode list in WordPress editor <strong>TinyMCE</strong>.
8. Add Shortcode by <strong>Visual Composer</strong>.
9. Add custom css class in block for customization.
10. Add custom css class in block by <strong>Visual Composer</strong> for customization.

== Installation ==

1. Upload `Find Any Think` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Simply download and upload / Installed and active the plugin, after activation you will find MKBlock Option in your admin panel, for more information please watch video.
